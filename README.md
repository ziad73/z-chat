# z-chat 
A simple, free chat application where you can create private chat rooms and share them with friends.

## How to Use

### Creating a Chat Room
1. Enter a unique room name (e.g., "GameNight", "StudyGroup")
2. Click **"Create"**
3. Share the room name with your friends

### Joining a Chat Room
1. Type the **exact room name** in the search box
2. The room will appear if it exists
3. Click on it to join and start chatting

### Chatting
1. Enter your name (or stay as "Anonymous")
2. Type your message
3. Press **Enter** or click **Send**
4. Everyone in the room sees your messages in real-time!

## Privacy

- All chat rooms are **hidden by default**
- Only people who know the **exact room name** can find and join
- Perfect for private group conversations

## Features

- ✅ Create unlimited chat rooms
- ✅ Real-time messaging
- ✅ Search by exact room name
- ✅ See message count for each room
- ✅ No sign-up required
- ✅ 100% Free

## Tips

- Choose unique room names to keep your chats private
- Room names are case-insensitive ("MyRoom" = "myroom")
- Share room names securely with only people you want to chat with
- Use the "Back" button to return to the room list

## Technical

- Built with HTML, CSS, and JavaScript
- Powered by Firebase Realtime Database
- Hosted on GitHub Pages
- No backend server required
